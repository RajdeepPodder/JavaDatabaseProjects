USER INSTRUCTIONS:

In this project I have created a Movie Database with a combination of  Java and oracle SQL as the database. The project will be simple and focused towards to the implementation of various database operations. It will take in the user’s input/request and query the database and give back the results.
INTRODUCTION:-


Movie Database consists of a lot information about the movies which helps the users to know the required details before the watching or buying tickets for that movie. The database consists of information like Movie name, Movie description, runtime, release date of the movie, genre of the movie, information about the cast, directors and most importantly ratings given by the users who have already watched it before. Movie database might also contain information about the availability of tickets. 

PROBLEM STATEMENT AND DISCUSSION:-


The goal of this project is to build a Movie Database which can help users to :
•	Find new movies to watch
•	Rate movies
•	See information about directors, cast and many more.
•	Admin can add or delete a movie.
•	Get information about runtimes and release dates
